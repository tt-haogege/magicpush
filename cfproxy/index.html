CFProxy

MagicPush 专用转发服务

📖 项目用途

本服务专为 MagicPush 项目提供，部署在 Cloudflare Pages 上，用于解决 IPv6-only 服务器无法被 IPv4-only 用户访问的问题。

由于 Cloudflare 支持双栈 IP 协议（同时支持 IPv4 和 IPv6），本代理可将来自 IPv4 用户的请求转发至仅支持 IPv6 的 MagicPush 后端服务器，实现无缝访问。

⚡ 工作原理

用户请求 → Cloudflare Edge (双栈) → IPv6 回源 → 您的服务器

Cloudflare Pages Workers Functions IPv6 回源
